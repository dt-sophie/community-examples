# GitHub Dependabot Alert Triaging

**Automate, verify, and prioritize security vulnerability remediation using Dynatrace runtime intelligence and GitHub workflows.**

This use case provides all the essential artifacts and configuration required for deployment on both GitHub and Dynatrace. By integrating Dynatrace Application Security insights with your GitHub repository, you can triage, validate, and resolve Dependabot alerts with precision—directly in your development workflows and observability platform.

Key features:
- AI-assisted vulnerability verification using Dynatrace runtime analytics.
- Automatic dismissal of alerts for unused code and libraries.
- Comments with rich risk and entity context for confirmed vulnerabilities.
- Human-in-the-loop reviews ensure every automated remediation meets your organization’s standards.

Everything you need—from workflow and configuration files for GitHub Actions, to integration and automation steps for Dynatrace—is included to enable production-grade vulnerability triage based on real runtime intelligence.
