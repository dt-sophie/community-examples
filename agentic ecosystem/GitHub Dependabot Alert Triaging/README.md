# GitHub Dependabot Alert Triaging

**Automate, verify, and prioritize security vulnerability remediation using Dynatrace runtime intelligence and GitHub workflows.**

By integrating Dynatrace Application Security insights with your GitHub repository, you can triage, validate, and resolve Dependabot alerts with precision—directly in your development workflows and observability platform.
This use case package provides all the essential artifacts and configuration required for deployment on both GitHub and Dynatrace. 

Key features:
- AI-assisted vulnerability verification using Dynatrace runtime context.
- Automatic dismissal of alerts for unused code and libraries.
- Automated fixes for confirmed vulnerabilities.
- Human-in-the-loop reviews ensure every automated remediation meets your organization’s standards.
