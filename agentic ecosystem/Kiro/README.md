# Kiro Agent Instructions for Dynatrace

This folder contains instruction files for the Kiro agent to interact with Dynatrace.

## Files

- `dynatrace-instructions.md`: Comprehensive guide and instructions for the agent to use Dynatrace MCP tools, including DQL querying, entity finding, problem management, and more.

## Usage

These instructions are intended to be provided to the Kiro agent to enable it to effectively monitor and manage Dynatrace environments.
