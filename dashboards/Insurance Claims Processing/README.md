 # Insurance Claims Processing Dashboard

This dashboard focuses on a business observability use case, visualizing the performance and status of an insurance claims process. 

<img width="1800" height="1000" alt="Screenshot 1" src="https://github.com/Dynatrace/community-examples/blob/main/dashboards/Insurance%20Claims%20Processing/Screenshot%20-%20insurance%20claims%20processing.png" />

## 🎯 Purpose

The dashboard helps you quickly understand **how well services perform from an end-user perspective** within your application.  
It categorizes requests into **Satisfied**, **Tolerating**, or **Frustrated** groups and visualizes Apdex trends over time.

## 📊 What You’ll See

This dashboard includes different types of tiles to help interpret service performance:
- **KPIs** – single value tiles with conditional formatting
- **Location** – visualizing business events on a geo map
- **Information over time** - plot timeseries or single value with tiles such as scatterplot, graph or bar charts for additional insights
