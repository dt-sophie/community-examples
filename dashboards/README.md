# 📊 Dynatrace dashboard examples

Welcome to the **Dynatrace dashboard examples** folder!  
This space is dedicated to sharing, exploring, and contributing **real-world dashboard configurations** built with Dynatrace.
Whether you're a Dynatrace user, partner, or customer, you're invited to contribute dashboards that showcase best practices, creative visualizations, or helpful monitoring setups.

---

## 🎯 Purpose

This folder serves as a **community-driven collection** of dashboards to help others:
- Get inspired by real use cases
- Learn how to visualize data effectively
- Accelerate setup for common scenarios

## 🚀 Getting started

To get started with Dynatrace dashboards, check out the [app spotlight](https://www.youtube.com/watch?v=K8rcj7fK9NM.) or dive into the [documentation](https://docs.dynatrace.com/docs/shortlink/dashboards). 
To try the examples shared here, simply upload them to your own environment. 
Don't have an account or environment yet? [Sign up](https://www.dynatrace.com/signup/playground/) and start experimenting in the Dynatrace [playground](https://playground.apps.dynatrace.com/) instead.

## How to contribute?

To contribute, follow these steps:
1. Submit a Pull Request: All contributions must be made via a pull request. 
2. Review Process: Pull requests will be reviewed by project owners and a selected group of Dynatrace experts to ensure quality and consistency.

### Contribution tips
* Use Descriptive Titles: Include a clear and concise name, title, and short introduction for your dashboard.
* Focus on Immediate Value: Ensure the dashboard provides immediate value and demonstrates the power and simplicity of Dynatrace.
* Follow Best Practices:
** Avoid errors and configure settings correctly.
** Use responsive layouts and WCAG-compliant colors.
** Include relevant data above the fold for better usability.
** Start from an Example: Use existing dashboards as a reference to align with quality standards and best practices.

By adhering to these guidelines, you’ll help create dashboards that are both useful and inspiring for the Dynatrace community.
